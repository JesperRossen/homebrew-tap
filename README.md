# homebrew-tap

Homebrew tap for [version-check-mcp](https://github.com/JesperRossen/version-check-mcp) - a lightweight MCP server that gives AI coding agents a single source of truth for software package versions.

## Install

```sh
brew install JesperRossen/tap/version-check-mcp
```

Formulas in this tap are published automatically by GoReleaser on each release.
